# Extrapolate（外推 / 推算）

> 👉 基于已有数据或趋势，向“未观察到的范围”延伸

==核心含义==
- 有明确的数据或模型
- 假设现有规律继续成立
- 常见于数学、统计、经济、工程、机器学习
==例子==
- 根据过去五年的数据，外推（extrapolate） 明年的经济增长率
- 该模型可以 extrapolate 到更大的输入规模

📌 如果趋势本身会变化 → 外推可能失效
# Infer（推断）

> 👉 根据证据和线索，得出合理结论

==核心含义==
- 有证据，但结论不是直接给出的
- 属于逻辑推理
- 强度通常 强于 speculate，弱于直接证明
==例子==
- 从日志信息中，我们推断（infer） 问题出现在初始化阶段
- 她从语气中推断 他心情不好

📌 infer 是最“学术安全”的词之一
