# ETH Zürich - Principles of Macroeconomics (with Chinese Annotations)

Course notes for **Principles of Macroeconomics** at ETH Zürich. English as the primary language, with key points emphasized in Chinese (中文重点标注).

I took this course as my **Science in Perspective** elective — it's very suitable for Science in Perspective. Written for exam review during my Computer Science master's — I'm not an economics major, so some explanations may be informal or simplified. Final grade: **5.75/6.0**, so these notes might actually be useful!

## Usage

This is an [Obsidian](https://obsidian.md) vault. To use it:

1. Clone this repository
2. Open the folder as a vault in Obsidian
3. The `ANKI/` folder contains Anki flashcards. The vault includes the [Yanki](https://github.com/kitschpatrol/yanki-obsidian) plugin — enable it in Obsidian to sync these flashcards to your [Anki](https://apps.ankiweb.net) app

## Structure

- **Homepage.md** — Obsidian's Entry point
- **Knowledge/** — Course notes
- **Glossary/** — Key economic terms and definitions
- **ANKI/** — Flashcard-style Q&A for active recall
- **Cengage MCQs/** — Official MCQ practice from the course textbook (PDF)
- **Documents/** — PPT slides and textbook used during exam review (removed for copyright reasons; add your own)
